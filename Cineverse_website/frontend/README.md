# CineVerse Frontend

Modern React application for CineVerse movie booking platform.

## Tech Stack
- React 18
- Vite
- React Router
- Axios
- JWT Decode

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment

Create `.env`:
```
VITE_API_URL=http://localhost:8080/api
```

See root README.md for full documentation.
